> [!WARNING]
> JSRinth is still in early development, and should probably not be used in production.

## Contributing

We use `pnpm` instead of `npm` in this project. It can be installed with `npm i -g pnpm`.

### Install dependencies

Run `pnpm i`

### Running test

Run `pnpm test`

### Running formatter

Run `pnpm format`

### Running linter

Run `pnpm lint`
